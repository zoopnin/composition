The problem at hand...

As programmers we find ourselves spending weeks and sometimes months writing web applications which under the hood, essentially all do the same thing and we've had enough, we cant take it anymore!
"Save this or get that from the database, put this or get that from the file system, send that email if this happens, resize this image, convert that video. click this do that."
Grrrr, no more we say!
"Give me an API end point for this"
No more, seriously!

We're building you non technical founders, architects and front end developers an intuitive, web based, visual scripting system so you can build even the most intricate of web apps quickly and without any programming. Connect together objects in virtually limitless combinations to assemble world class web apps yourselvs.

Yes, yes, it's going to scale infinitely and yes its going to be hosted on AWS, and no it wont be impossible to figure out or be expensive to get started.

You just need to drag and drop objects, wire them up with connections. Groups of connected objects can then be wrapped into a single object and again wired up to further objects, as a non programmer its stupidly powerful and empowering.

Front end designers, don't panic we still need you to make it look all pretty, we're not touching the presentation layer,
We will give you working HTML and JavaScript, you still get to to style it up and do all your fancy design bits, it's still all on you, to make it look cool.

"but It wont be able to do this! or that! No way!"
Listen if you really want to do some weird stuff, you can! give us a call or find a programmer to build you a custom object, then you can load it in re-use it over and over again and wire it up with your existing objects.
